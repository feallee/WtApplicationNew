# WtApplication 简介
WtApplication 是基于操作系统的用户框架，特别适合单片机级开发实践。提供启动入口，休眠入口，唤醒入口，命令等管理接口。
# 推荐工程框架
```TABLE
+-------------------------------------------------------+
|              APP(CONSOLE,NET,GPS,BT,...)              |
+-------------------------------------------------------+
+-----------------------------------------------+   +---+
|             BSP(PIN,UART,SPI,...)             |   |   |
+-----------------------------------------------+   |   |
+-----------------------+   +-----------------------+   |
| LIB(STM32,WCH,FM,...) |   | OS(RT-Thread/FreeRTOS/... |
+-----------------------+   +---------------------------+
```
